# Запуск

`npm i`

`npm run start`

Далее скопировать содержимое `.env.example` в `.env` 

# Структура проекта

`src` - папка с исходным кодом

`cypress` - интграционные тесты

`src/slices` - папка с слайсами Redux (включая unit тесты)

`src/services` - папка с инициализацей стора Redux

`src/actions` - middleware для Redux (Redux-thunk)

`src/components` - папка с компонентами React

`src/utils` - папка с утилитами (включая API)

`src/pages` - папка со страницами
